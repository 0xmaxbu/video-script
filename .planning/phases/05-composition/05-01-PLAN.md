# Phase 5-1: Quality Settings - CRF, Retina, and Composition ID

## Goal

Fix video quality settings: add CRF 20 to Remotion CLI, set deviceScaleFactor to 2 for Retina screenshots, and add compositionId support for dual-resolution rendering.

## Context

From research:
- **video-renderer.ts** spawns Remotion CLI without `--crf` flag (uses default CRF 23)
- **puppeteer-renderer.ts** uses `deviceScaleFactor: 1` (blurry Retina screenshots)

## Requirements

- COMP-02: Video quality feels polished and professional

## must_haves

- CRF 20 is passed to Remotion CLI (video-renderer.ts)
- CRF 20 is passed to FFmpeg stitch (puppeteer-renderer.ts)
- deviceScaleFactor is set to 2 for Retina screenshots (puppeteer-renderer.ts)
- compositionId parameter is supported in video-renderer.ts for selecting portrait/landscape composition
