Purpose: Text overflow is one of the most visible quality issues in generated videos. fitText computes the largest font size that fits within a container width, ensuring professional presentation across variable content. Satisfies VIS-14.

Output: TextLayer, BulletList, HeroFullscreen updated with fitText. `@remotion/layout-utils` installed if not present.

      text: "Test",

      fontFamily: "system-ui",
